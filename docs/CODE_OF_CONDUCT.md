# Code of Conduct

We are committed to fostering a welcoming and inclusive environment for everyone, regardless of background or identity.
We expect all contributors to treat others with respect, use professional language, and accept constructive criticism gracefully.
Harassment, hate speech, trolling, and personal attacks are strictly prohibited and will not be tolerated.
Project maintainers reserve the right to remove comments or ban users who violate these community standards.
If you witness or experience unacceptable behavior, please report it to the project maintainers immediately.
By participating in this project, you agree to abide by these guidelines.
