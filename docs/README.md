# Crowd Monitoring Detection and Monitoring System

This is a simple crowd monitoring and cluster detection system with a heatmap developed using O>

## Overview
This is a simple python script which uses the laptop webcam to detect people in real time with >

**Tech Stack**:
- Python
- numpy (For creating arrays of detected objects)
- OpenCV (for computer vision, object detection, and image processing)
- DBSCAN (For detecting cludters of people - module from scikit-learn)

## Steps for Installation and Running

1. Clone this repository on your local machine by running git clone:
```bash
git clone
```

2. Navigate inside the repository
```bash
cd Crowd-Detection-and-Monitoring-System
```

3. Run the python script crowd_detect.py
```bash
python crowd_detect.py
```

## Features
- OpenCV for crowd detection - counting people
- Heatmap overlay in real time
- Count of clusters of people based on object proximity logic
